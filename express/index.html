Selector Portal

We would like to crowdsource annotations for a collection of dog images using this AI and Society class. We will do this by first asking classmates for a collection of features, and then use a predictive machine learning algorithm to only choose classmates who, according to the algorithm, are likely to correctly label the images that we assign them. We really wish to do this right. Consequently, we are also asking for an opinion from many classmates about which of the four algorithms we developed we should choose.
